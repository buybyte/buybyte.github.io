
# name id pairs

> 对于 OS 来说，id 更重要。

| symbolic name(human readable name)  | number name(low level name)| 
| :--------: | :------: |
| service name |   port   |
| user name          |   user id | 
| process name  |  process id   | 
| host name |  ip address   | 
| file name |  inode number |
| signal name | signal number |
| syscall name | syscall number |
